## Como usar este diretório

Cada arquivo segue um modelo padronizado com os campos:

- **Problema** — o que estava errado
- **Causa** — por que aconteceu
- **Solução** — como foi resolvido
- **Comandos utilizados** — referência rápida
- **Referências** — fontes consultadas

---

## Template

Ao registrar um novo problema, copie o modelo abaixo:

```markdown
# Problema

## Descrição

## Causa

## Solução

## Comandos utilizados

## Referências
```
